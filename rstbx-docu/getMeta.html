Description

This is an accessor function to quickly access information stored in ImageMetaData, e.g. scale factor per band. Intended for use with imagery which was imported using stackMeta.

Usage

getMeta(img, metaData, what)

Arguments

img

Raster* or character vector with band names.

metaData

ImageMetaData or path to meta data file.

what

Character. Slot to extract.
