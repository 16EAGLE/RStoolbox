Description

Spectral Angle Mapper

Usage

sam(x, em, angles = FALSE)

Arguments

x

RasterBrick or RasterStack. Remote sensing imagery (usually hyperspectral)

em

Matrix containing endmembers in rows and bands in columns

angles

Logical. If TRUE a RasterBrick containing each one layer per endmember will be returned containing the spectral angles.

Value

RasterBrick or RasterLayer If FALSE a single Layer will be returned in which each pixel is assigned to the closest endmember class.
