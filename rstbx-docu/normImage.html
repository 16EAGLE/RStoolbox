Description

Normalize / Center an image or image stack.

Usage

normImage(img, norm = TRUE, ...)

Arguments

img

Raster*. Image to transform. Transformation will be performed separately for each layer.

norm

Logical. Perform normalization in addition to centering.

...

further arguments passed to writeRaster.

Value

Returns a Raster* with the same number layers as input layers with each layer being centered and optionally normalized.

Examples

data(rlogo)
# Normalization
normImage(rlogo)

#> class       : RasterBrick 
#> dimensions  : 77, 101, 7777, 3  (nrow, ncol, ncell, nlayers)
#> resolution  : 1, 1  (x, y)
#> extent      : 0, 101, 0, 77  (xmin, xmax, ymin, ymax)
#> coord. ref. : +proj=merc 
#> data source : in memory

#> min values  : -2.443678, -2.535155, -2.736829 
#> max values  : 0.9747949, 0.9526321, 0.8828542

# Centering
normImage(rlogo, norm = FALSE)
