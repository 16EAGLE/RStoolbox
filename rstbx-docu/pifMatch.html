Description

Automatically estimates pseudo-invariant-features (PIF) by pixel similarity and fits a linear model which is then applied to the whole scene.

Usage

pifMatch(img, ref, method = "cor", quantile = 0.95)

Arguments

img

Raster*. Image to be adjusted.

ref

Raster*. Reference image.

method

Method to calculate pixel similariry

quantile

Numeric. Threshold quantile used to identify PIFs
