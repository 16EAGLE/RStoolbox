Description

ENVI spectral libraries consist of a binary data file (.sli) and a corresponding header (.hdr, or .sli.hdr) file. Spectra are read into a data.frame with the columns: Wavelength, Spectrum 1, Spectrum 2 ...

Usage

readSLI(path)

Arguments

path

path to spectral library file with ending .sli.

References

ENVI header format

See Also

writeSLI

Examples

## Create fake spectra
data <- data.frame(wavelength=350:2500, spectrumA=cumsum(abs(rnorm(2151))),
                        spectrumB=cumsum(abs(rnorm(2151))))
pathToFile <- paste0(tempdir(),"/specLib.sli")

## Write to binary spectral library
writeSLI(x = data, path = pathToFile)

## Read from binary spectral library
dataRe <- readSLI(path = pathToFile)

## Check whether they are the same
all.equal(data, dataRe)

#> [1] "Names: 1 string mismatch"

## oops? nope, only colnames mismatch. numbers are fine.
cat("colnames:\noriginal: ",colnames(data),"\nre-read: ",colnames(dataRe))

#> colnames:
#> original:  wavelength spectrumA spectrumB 
#> re-read:  wavelengths spectrumA spectrumB
